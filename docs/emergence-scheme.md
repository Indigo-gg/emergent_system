# Ecosystem Emergence Explorer — 完整方案

## 一、系统概述

多物种粘菌（Physarum）生态系统模拟，通过 LLM 引导的参数搜索，自动发现涌现现象。

### 模拟对象
- **Explorer**（橙金）：速度 2.0，广感知，高代谢（0.015/tick），擅长发现新营养源
- **Grazer**（翠绿）：速度 0.8，低代谢（0.004/tick），高效利用，贫瘠区存活
- **Harvester**（青蓝）：速度 1.3，平衡型，营养提取率最高（0.8/单位）

### 跨物种信息素交互

每个物种不仅感知自己的信息素，还感知其他物种的信息素，带有正（吸引）或负（排斥）权重：

| 观察者 ↓ \ 被感知 → | Explorer | Grazer | Harvester |
|---------------------|----------|--------|-----------|
| **Explorer** | 自身 | +0.3（偏利共生） | -0.2（领地排斥） |
| **Grazer** | -0.1（轻微排斥） | 自身 | +0.4（追随觅食） |
| **Harvester** | +0.2（吸引） | -0.1（轻微排斥） | 自身 |

这创造了复杂的宏观结构：Explorer 开路 → Harvester 跟进 → Grazer 捡漏 → Explorer 被 Harvester 排斥形成领地边界。

### 个体生命周期（每 tick）
```
感知 → 转向 → 移动 → 沉积信息素 → 摄取营养 → 代谢消耗 → 死亡判定 → 繁殖判定
```

### 反灭绝机制
- **休眠孢子（Spore）**：能量耗尽时进入休眠状态（不移动、极低代谢 0.001/tick），附近有营养时苏醒
- 最大休眠时间 600 tick（10 秒），超时则真正死亡
- 承载力上限（繁殖减速）

### 废弃物/毒素系统
- 个体进食时产生废弃物（比例由 wasteProductionRate 控制）
- 废弃物也会扩散和衰减
- 高废弃物区域：个体代谢成本倍增 + 主动回避
- 涌现效果：聚集→过度消耗→废弃物爆发→环形逃离→脉冲式扩张收缩（类似 B-Z 化学振荡）

### 移动营养热点
- 营养注入中心沿螺旋路径缓慢漂移
- 创造"季节变化"——生态网络必须追逐营养迁移
- Explorer 前线探路 → Harvester 中段狂吃 → Grazer 尾部捡漏

## 三、探索策略（三阶段）

### Phase A — 拉丁超立方采样（40 轮）

使用 Latin Hypercube Sampling（LHS）而非纯随机。LHS 将每个参数范围分成 40 个等距区间，每个区间恰好采样一次，确保 13 维空间的均匀覆盖。

**LHS vs 纯随机：**
- 纯随机在 13 维空间中 40 个点大概率扎堆
- LHS 保证每个参数的每个值域都被探索
- 40 轮 LHS 的覆盖效果 ≈ 纯随机 100+ 轮

**目的：**
- 高效覆盖参数空间
- 建立 score-params 初始映射
- 发现参数空间中的"沙漠"（低分区）和"绿洲"（高分区）

### Phase B — LLM 引导搜索（30 轮）

MiMo v2.5 分析历史实验，智能建议下一组参数。

**LLM 收到的信息：**
1. 参数空间定义
2. 最近 10 轮实验结果（参数 + 得分 + 评语）
3. 历史最佳 3 轮
4. 相变边界检测（相邻高分差 > 3 分且参数相近）

**LLM 输出：**
```json
{
  "reasoning": "分析思路（100字）",
  "params": { ... },
  "hypothesis": "预期现象（50字）"
}
```

**引导策略：**
- 围绕高分区域做精细采样
- 检测到相变边界时做极小步长扫描
- 避免重复已试过的参数组合
- 探索未覆盖的参数区域

### Phase C — 遗传算法（50 轮）

取高分参数做"亲本"，变异产生下一代，优胜劣汰。

**算法流程：**
```
初始化：取 Top-5 高分实验作为精英种子
循环：
  1. 评估当前种群（每轮评估一个个体）
  2. 一代评估完后，进化：
     - 精英保留（Top-3，得分 ×0.95 衰减防过拟合）
     - 锦标赛选择（随机3个取最优）
     - 变异（每个参数 15% 概率变异，变异幅度 ±10% 范围）
  3. 重复
```

**种群规模：** 10 个个体/代
**精英数：** 3 个
**变异率：** 15%
**变异幅度：** 参数范围的 ±10%

## 四、评分规则

每轮模拟完成后，截图发给 MiMo v2.5 评分。

### 4.1 输入给 LLM 的内容

**Sprite Sheet 拼图：**
- 左侧 2×2 网格：模拟 70%、80%、90%、100% 时刻的截图
- 右上：最终 80 帧的累积轨迹图
- 右下：差异图（Δ Map）— 70% vs 100% 像素差分，蓝色=稳定，红色=变化

**差异图作用：**
- 大面积蓝色 → 系统已稳定，结构固化
- 大面积红色 → 系统仍在剧烈变化
- 蓝底红色纹理 → 稳定结构上有微流动（Dynamic Complexity 的关键证据）

**硬数学指标（自动计算）：**
- 各物种种群数和占比
- 物种均衡度（0-1，1=完全均衡）
- 种群趋势（rising/stable/falling）
- 种群方差

### 4.2 评分维度（6 个，每项 1-10 分）

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| **Species Coexistence** | 0.20 | 1分=一个物种完全主导；5分=三物种都存活但不均；10分=稳定均衡共存 |
| **Spatial Self-Organization** | 0.20 | 1分=均匀随机；5分=可见领地划分；10分=多层级空间结构+复杂边界 |
| **Nutrient Network** | 0.15 | 1分=营养随机分散；5分=通道雏形；10分=类似血管网络的自组织结构 |
| **Structural Stability** | 0.15 | 1分=四张切片完全不同；5分=3张切片中结构持续；10分=完全稳定 |
| **Dynamic Complexity** | 0.15 | 1分=简单扩散或静止；5分=可见流动/脉动；10分=多尺度动力学 |
| **Aesthetic Emergence** | 0.15 | 1分=视觉噪声；5分=有吸引力；10分=让人屏息的有机生态景观 |

### 4.3 加权计算

```
composite_score = Σ(维度分 × 权重)

= coexistence × 0.20
+ organization × 0.20
+ network      × 0.15
+ stability    × 0.15
+ complexity   × 0.15
+ aesthetic    × 0.15
```

### 4.4 硬约束（强制扣分）

| 条件 | 约束 |
|------|------|
| 某物种占比 > 80% | coexistence 最高 3 分 |
| 某物种占比 < 5% | coexistence 最高 4 分 |
| 总体种群持续下降 | 额外警告，可能影响多个维度 |
| 营养场完全空白 | 总分最高 2 分 |
| 所有个体堆积角落 | 总分最高 2 分 |
| 数值溢出异常图案 | 总分最高 1 分 |

### 4.5 涌现等级

| 分数区间 | 等级 | 描述 |
|----------|------|------|
| < 3.0 | none | 无涌现 |
| 3.0 ~ 5.0 | weak | 弱涌现，初步自组织 |
| 5.0 ~ 7.0 | moderate | 中等涌现，可见结构 |
| 7.0 ~ 8.5 | strong | 强涌现，稳定自组织 |
| > 8.5 | extraordinary | 卓越涌现，类似真实生态系统 |

### 4.6 新颖性判定

LLM 对比最近 5 轮实验描述，如果当前形态与所有历史截然不同：
- `is_novel = true`
- 附带 `novelty_note` 描述新颖之处

## 七、涌现理论基础

### 为什么用 LLM 评分而非纯数学？

涌现的核心特征是"整体大于部分之和"——这难以用单一指标量化。

纯数学指标的局限：
- 同步序参量 R → 只衡量同步，不衡量结构
- 簇数量 → 只衡量聚集，不衡量复杂性
- 信息熵 → 衡量随机性，不衡量"有意义的模式"

LLM 的优势：
- 视觉模式识别 → 能判断"看起来像生态系统"
- 跨维度综合 → 同时考虑结构、动力学、美学
- 类比推理 → 能识别"类似血管网络"这样的隐喻

### 相变边界检测

涌现通常发生在参数空间的"相变边界"——有序与混沌的刃尖上。

检测方法：相邻实验参数相近但得分差异 > 3 分 → 可能是相变边界。

应对策略：围绕边界做极小步长采样（step/4），精确定位涌现窗口。
